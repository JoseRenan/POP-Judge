ConTESTE POPJudge | P2

Vanlex pede socorro!

Por José Renan, IFPB - Campina Grande Brazil

Timelimit: 1

Valenx é um professor de matemática do IFEFPB (Instituto Federal de Ensino Fundamental da Paraíba). Vanlex percebeu que seus alunos estao com dificuldades em operações básicas da matemática e pediu para os alunos de informática do IFPB para fazer um programa que resolvesse calculos simples de operações básicas.

Entrada

A entrada consiste em vários casos de teste que contem um inteiro X um operador O (' + ', ' - ', ' / ', ' * ' ) e outro inteiro Y. (0 < X ,Y < 100). A entrada termina com o número 0.

OBS: Os resultados não inteiros devem ser arrendondados para baixo.

Saída

A saída consiste no resultado das expressões apresentadas.

Exemplo de Entrada Exemplo de Saída

1/2

1+1

1*1

1-9

0

0

2

1

-8
