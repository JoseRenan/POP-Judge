InfoOS

Por José Renan, IFPB - Campina Grande Brazil

Timelimit: 2

A turma de informática do IFPB criou um novo sistema operacional, o InfoOS. Com ele é possível usar qualquer número de caracteres '/' como delimitador de caminhos de diretórios. Por exemplo, o caminho: '//usr///local//nginx/sbin//' é a mesma coisa que '/usr/local/nginx/sbin'. Os alunos de informática precisaram passar algumas pastas para o ubuntu, no entanto as pastas estavam com o caminho com vários '/' delimitando os diretórios, sua missão é ajudá-los a colocar os caminhos de forma que funcione em ambos os sistemas operacionais.

Entrada

A entrada consiste em vários casos de teste e é encerrado pelo número 0

Saída

A saída consiste no caminho mostrado de forma que seja aceito por ambos SOs.

Exemplo de Entrada Exemplo de Saída

//usr///local//nginx/sbin

////a//b/////g

/a/b/c

/a//aa/a//

0

/usr/local/nginx/sbin

/a/b/g

/a/b/c

/a/aa/a
